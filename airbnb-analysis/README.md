# Airbnb Listings Analyzer (Python)

## Overview
This project analyzes Airbnb listing data to explore what factors relate to price and to summarize patterns in the market.

## Skills & Tools
- Python (pandas, numpy, matplotlib)
- Data cleaning (missing values, formatting, filtering)
- Exploratory data analysis (EDA) + visualization

## What I Did
- Loaded and cleaned the dataset (handled missing values and inconsistent formats)
- Created summary statistics and visualizations for key variables
- Explored relationships between price and listing features (e.g., bedrooms, bathrooms, location, ratings)

## Key Outputs
- Visualizations of price distribution and feature relationships
- Summary tables showing trends by feature groups

## Files
- Notebook: `notebooks/airbnb_analysis.ipynb`
